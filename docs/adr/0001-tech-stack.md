# ADR 0001: Tech stack selection

## Decision
Use Next.js (TypeScript) for frontend, FastAPI (Python) for backend, PostgreSQL (Supabase) for database.

## Rationale
- FastAPI aligns with data science skillset and ML integration.
- Next.js provides modern UI and easy free deployment.
- PostgreSQL is production-grade and supported by Supabase free tier.

## Status
Accepted
